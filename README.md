# openapiexample2
Welcome to the openapiexample2

El presente es un ejemplo de como usar openapi con Redoc.

* descargar todo el projecto.
* ejecutar los siguientes comandos:
  `npm install`
  `npm run build`
  `npm run start`

* ingresamos la url: http://127.0.0.1:8080/
